# The Final Countdown

A single-page, poetic street-art tribute counting down to the last hour of an eternal war —
the moment the guns fall silent.

**Ceasefire:** 23:00 &middot; 23 July 2026 &middot; Zürich time (CEST, UTC+2)

## The piece

A Banksy-flavoured concrete wall at the edge of dawn: stencilled type, a spray-paint
dove carrying an olive branch, dripping red paint, drifting embers, and a huge live
countdown. When the clock reaches zero, the wall stops counting the dead and simply
reads *Peace.*

It is a single self-contained `index.html` — no build step, no dependencies beyond
Google Fonts. Just open it in a browser.
